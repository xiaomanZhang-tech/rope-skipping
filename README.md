功能核心设计
1 核心玩法
一句话概括：用户启动一分钟跳绳功能后，APP通过摄像头捕捉人体跳跃动作，自动计时（60秒）、精准计数，结束后生成详细运动数据，支持打卡、分享、查看历史记录，适配训练、打卡场景；全程保障视频流畅，无卡顿、掉帧现象。
详细流程：
      1.  用户进入天天跳绳APP首页，点击“计时跳绳”，选择“一分钟模式”，进入功能准备界面；
      2.  按照界面指引，授予APP相机权限，将手机放置在能完整捕捉人体全身的位置（如地面、桌面），调整角度确保身体在识别范围内；
      3.  点击“开始”按钮，系统启动60秒倒计时，同时AI开始捕捉跳跃动作，实时计数并同步显示，此时相机预览、AI识别、计时计数同步运行，确保视频流畅无卡顿；
      4.  运动过程中，用户正常跳绳（普通跳绳、空手模拟均可），APP实时过滤无效动作，仅计数有效跳跃，同时维持视频流畅，避免因AI识别负载过高导致卡顿；
      5.  倒计时结束后，自动停止计数，弹出结算界面，展示本次一分钟跳绳总跳数、平均每秒跳数、消耗卡路里等数据，界面切换流畅无延迟；
      6.  用户可选择“重新挑战”“打卡”“分享”“查看历史”等操作，完成本次训练，所有操作响应及时，无卡顿。
    
2 功能整体流程
APP首页 → 计时跳绳入口 → 一分钟模式选择 → 相机权限授予 → 设备摆放指引 → 开始计时计数（保障视频流畅） → 运动过程（实时反馈，无卡顿） → 倒计时结束 → 结算数据展示 → 后续操作（重新挑战/打卡/分享/历史记录）
3 功能规则
•计时规则：默认计时60秒，点击“开始”后自动启动倒计时，实时显示剩余时间；倒计时最后10秒触发语音提示（“还有10秒”“3、2、1，时间到”），提升用户体验；计时过程中与相机预览、AI识别同步运行，无卡顿、无延迟。
•计数规则：
        
•有效跳跃判定：AI识别人体脚踝、膝盖等核心骨骼点，当脚踝垂直位移达到设定阈值（适配不同身高自动调节）、身体无明显倾斜，且完成一次完整的“起跳-落地”动作，判定为1次有效跳绳，计入总跳数；AI识别过程中优化算法，避免因负载过高导致视频卡顿；
•无效动作过滤：过滤轻微跳动、身体倾斜、无跳绳动作的摆动、卡绳停顿等无效动作，不计数，避免误差；同时优化过滤逻辑，减少处理器负载，保障视频流畅；
•计数精度：计数准确率≥97%，适配不同跳跃速度（慢跳至快速跳）、不同跳绳姿势（双脚跳、单脚跳），支持空手模拟跳绳计数，无需专用跳绳，普通跳绳即可使用；计数过程中不影响视频流畅度，避免卡顿、掉帧。
暂停/重新开始规则：运动过程中支持点击“暂停”，暂停后计时、计数均停止，恢复后继续计时；支持点击“重新开始”，重置计时（60秒）和计数（0次），重新启动运动；暂停、重新开始操作响应及时，无卡顿，恢复后视频仍保持流畅。
打卡规则：结算后点击“打卡”，自动记录本次成绩至用户历史记录，支持关联班级群（供学生提交作业），打卡成功后可获得对应积分或勋章（可选），激励用户持续训练；打卡操作响应及时，无卡顿。
